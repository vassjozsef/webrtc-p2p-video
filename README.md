Peer-to-Peer WebRTC Sample
==========================

WebRTC peer-to-peer video sample for testing transport-wide feedabck.

Deployment
----------

To run locally, use:

    yarn
    yarn start

To run on heroku:

   heroku create
   git push heroku master
   heroku open
